This plugin lets you build a way to collect post types into a report and print them nicely

== Description ==

A Plugin that lest you more easily create a report to build. 


== Installation ==

Regular installation + adding some shortcodes in different places. 

To get it to work:
1. Add the  [add_to_report] shortcode to display where you want the button to show. 
You don't want more then one to display on the page

2. Create a page that will have the help user to print it.
and add the shortcode [display_report]

3. Add the [report_list print_url="print_report_page_url"] to display where you want it to.
